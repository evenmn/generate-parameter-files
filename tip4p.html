The TIP4P force fieldÔÉÅ

The TIP4P force field object differs from the StillingerWeber and Vashishta objects in the sense that a parameter file is not written, but rather a LAMMPS script that can be included in the LAMMPS input script. Apart from that, the usage is similar to the other force field objects:

from genpot import TIP4P

potential = TIP4P("tip4p-2005")

Since TIP4P always contains one molecule type (water), interaction groups are never specified. The possible parameters are Z_H, Z_O, r0, theta, OM, epsilon, sigma and r4s. They can be modified by

potential.update_params({'epsilon': 0.1855, 'sigma': 3.16})

As for the Vashishta-object, the charges are considered as coupled such that the molecule is neutral. Therefore, changing Z_H would also change Z_O. The LAMMPS script is written to file using the write-method:

potential.write("tip4p.water")
